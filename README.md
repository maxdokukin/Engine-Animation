# Engine Animation

Accurate cylinder animation for 4 stroke engine. 

Animates strokes:
Phase 1: The Intake Stroke. ...
Phase 2: The Compression Stroke. ...
Phase 3: The Power Stroke. ...
Phase 4: The Exhaust Stroke.

Additionally, valve movements, and sparks.
